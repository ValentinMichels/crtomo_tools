CRTomo user space tools and libraries.

Installation
------------

Quick linux install commands (refer to the crtomo-doc for detailed
instructions) ::

	mkvirtualenv --python /usr/bin/python3 crtomo
	pip install --upgrade pip
	pip install -r requirements.txt
	pip install .
